# Security Research and Information Centre (SRIC) - Kenya

This repository contains the web application for the Security Research and Information Centre (SRIC) in Kenya. SRIC is an independent non-profit think-tank committed to providing data and information on human security and security sector dynamics in Kenya and the sub-region.

## Tech Stack
- **React 19**
- **Vite**
- **TypeScript**
- **Framer Motion** for animations
- **React Router** for routing
- **React Helmet Async** for SEO

## Project Structure
- `/src/components` - Reusable UI, Layout, and Common components
- `/src/pages` - Route components representing specific pages
- `/src/hooks` - Custom React hooks
- `/src/types` - TypeScript interfaces and types
- `/src/utils` - Helper functions
- `/src/styles` - Global CSS styles and themes
- `/src/data` - Static data for the website

## Getting Started

### Prerequisites
Make sure you have Node.js and npm installed.

### Installation
1. Clone the repository
2. Install dependencies:
```bash
npm install
```

### Running the Development Server
```bash
npm run dev
```

### Building for Production
```bash
npm run build
```

### Linting & Formatting
```bash
npm run lint
npm run format
```
